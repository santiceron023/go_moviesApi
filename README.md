# go_moviesApi
### Back End Api

Go Api to create and list (by ID and All) Movies

## Características
* ***Hexagonal architecture***

* No sql DB
* MiddleWare for error log
* No ORM, faster Db access


## Tecnologías

***Data Base:*** Cassandra

***Dependency:*** Go modules 

***Testing:*** Docker test Containers
